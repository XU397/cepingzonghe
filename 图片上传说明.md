# 背景图片上传说明

## 问题说明
由于技术限制，AI无法直接保存您上传的二进制图片文件。当前页面使用的是CSS渐变模拟背景。

## 解决方案

### 步骤1：手动保存图片
1. 将您的光伏治沙背景图片重命名为：`pv-sand-background.jpg`
2. 手动复制到以下目录：
   ```
   C:\Users\LH\Desktop\cepingzonghe\src\submodules\g8-pv-sand-experiment\assets\images\
   ```

### 步骤2：修改代码使用真实图片
修改 `Page02Cover.tsx` 文件：

```typescript
// 在文件顶部添加import
import backgroundImage from '../assets/images/pv-sand-background.jpg';

// 在return语句中替换背景div
<div className={styles.backgroundImage}>
  <img 
    src={backgroundImage} 
    alt="光伏治沙背景"
    className={styles.backgroundImg}
  />
</div>
```

### 步骤3：更新CSS样式
在 `Page02Cover.module.css` 中将：
```css
.placeholderBackground {
  /* 当前的渐变样式 */
}
```

替换为：
```css
.backgroundImg {
  width: 100%;
  height: 100%;
  object-fit: cover;
  object-position: center;
}
```
