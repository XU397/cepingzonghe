# 交互式课件开发需求文档：任务C 光伏治沙

## 1\. 项目概况

  * [cite\_start]**项目名称：** 任务C：光伏治沙 [cite: 95]
  * **核心功能：** 通过SVG仿真实验，模拟光伏板对地表风速的影响，引导学生进行数据采集与分析。
  * [cite\_start]**技术规范：** 遵循《页面内容输出规范》Type-A 至 Type-F 标准 [cite: 2]。

## 2\. 核心数据配置 (Global Config)

**特别注意：** 本实验的SVG动画与图表渲染必须严格绑定以下离散数据，严禁使用随机数。数据来源为补充提供的《实验数据表》。

请在前端代码中定义如下常量（单位：m/s）：

```javascript
const WIND_SPEED_DATA = {
  // 20cm高度: 光伏板下风速更低
  20: { withPanel: 2.09, noPanel: 2.37 },
  // 50cm高度: 光伏板下风速更低
  50: { withPanel: 2.25, noPanel: 2.62 },
  // 100cm高度: 光伏板下风速显著降低 (关键趋势点)
  100: { withPanel: 1.66, noPanel: 2.77 }
};
```

### Page 06: 实验探究-2 (趋势分析)

  * [cite\_start]**页面类型：** **Type-E (实验+选择题混合页)** [cite: 43]
  * **布局配置：**
      * **左侧 (实验面板)：** 复用 SVG 组件。
      * **右侧 (答题区)：**
          * [cite\_start]**题目：** “问题3：根据实验模拟，关于风速随高度变化的描述，正确的是？” [cite: 172]
          * **选项 (Radio Group)：**
              * A. 在两区域，风速都随高度增加而增加
              * B. 在两区域，风速都随高度增加而减小
              * [cite\_start]C. 在无板区，风速随高度增加而增大；在有板区，风速随高度增加而减小 [cite: 180-181]
              * D. 在无板区，风速随高度增加而减小；在有板区，风速随高度增加而增大
  * **关键逻辑说明：**
      * 学生需自行测试 20cm 和 100cm。
      * **无板区趋势：** 2.37 -\> 2.62 -\> 2.77 (随高度增加而**增大**)。
      * **有板区趋势：** 2.09 -\> 2.25 -\> 1.66 (在100cm处显著下降，整体趋势视为**减小**)。
      * **正确答案：** C。
