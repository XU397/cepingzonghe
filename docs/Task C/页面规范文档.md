每种类型的信息模板包含
1. Type-A：基础信息展示页 
✅ 页面元数据（编号、标题、描述、导航模式）
✅ 内容结构（文字/列表/图片/复选框）
✅ 倒计时配置（时长、完成行为）
✅ 下一页按钮逻辑
样例：Page01_Notice（40秒倒计时 + 复选框确认）
2. Type-B：单一文本输入页 
✅ 左右分栏布局（左侧图片 + 右侧输入框）
✅ Textarea配置（行数、占位符、验证规则）
✅ 实时验证逻辑
✅ 提交格式
样例：Page03_Question（左侧对话图 + 右侧问题输入）
3. Type-C：多文本输入页 
✅ 多个输入框配置（id、label、验证）
✅ 综合验证规则（所有输入框至少N字符）
✅ 逐个字段的数据提交格式
样例：Page07_Design（3个想法输入，每个≥2字符）
4. Type-D：复杂交互选择页 
✅ 左侧模态弹窗按钮组（资料中心）
✅ 右侧复选/单选框组
✅ 模态窗口内容配置（图片/文本）
✅ 查看追踪逻辑
✅ 多选答案的提交格式
样例：Page04_Resource（5个资料弹窗 + 6个因素复选框）
5. Type-E：实验+选择题混合页 
✅ 左侧实验面板配置（复用IntegratedExperimentPanel）
✅ 右侧选择题配置（题目、选项、类型）
✅ 实验与答题的关联逻辑
✅ 双重数据提交（实验结果 + 答案）
样例：Page11_Analysis1（左侧继续实验 + 右侧单选题）
6. Type-F：SVG动画实验页 
✅ 量筒配置（数量、标签、颜色、高度）
✅ 温度控制配置（范围、步长、控制方式）
✅ 物理模型配置（函数名、输入输出）
✅ 动画参数（帧率、小球属性）
✅ 实验历史记录配置
✅ 图片资源清单（按钮图标）
样例：Page10_Experiment（4个量筒同时下落 + 温度控制）
7. Type-G：标准化问卷页 
✅ JSON数据结构规范（pageNumber、questions、options）
✅ 问题类型定义（likert4、likert5、binary）
✅ 自动化行为说明（无需手动编码）
✅ 极简页面组件（3行代码）
样例：Page15_Questionnaire1（完全由JSON驱动）

---
特别说明
🎯 Type-G（问卷页）的最大优势
只需提供JSON数据文件，页面组件只需3行代码：
import BaseQuestionnairePage from '../components/questionnaire/BaseQuestionnairePage';

const PageXX = () => <BaseQuestionnairePage pageNumber={14} />;

export default PageXX;
