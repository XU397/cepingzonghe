## **0\. 全局实验核心逻辑 (致AI工程师 \- 重中之重)**

本模块包含一个贯穿 Page 4-6 的核心实验模型。请**不要**使用通用物理公式计算，而是使用以下精确数据，以确保与教学课件完全一致。

### **A. 核心变量定义**

* **Height (飞行高度)**: 离散值 \[100, 200, 300\] (单位: 米)  
* **FocalLength (镜头焦距)**: 离散值 \[8, 24, 50\] (单位: 毫米)

### **B. 数据查找表 (Data Lookup Table)**

请在代码中构建如下映射关系，当用户选择对应 H 和 f 时，输出对应的 GSD 值：

| 飞行高度 (m) | 镜头焦距 (mm) | 地面采样距离 GSD (cm/px) |
| :---- | :---- | :---- |
| **100** | **8** | **3.01** |
| 100 | 24 | 1.00 |
| 100 | 50 | 0.48 |
| **200** | **8** | **6.03** |
| 200 | 24 | 2.01 |
| 200 | 50 | 0.96 |
| **300** | **8** | **9.04** |
| 300 | 24 | 3.01 |
| 300 | 50 | 1.45 |

### **C. 视觉反馈逻辑 (SVG/Canvas动画)**

GSD 数值直接控制结果图片的模糊程度。

* **逻辑：** GSD 值越大，图片越模糊（代表分辨率越低）。  
* **实现建议：** 使用 CSS filter: blur(X px)。  
* 映射算法参考： \`\`\`javascript  
  // 建议模糊系数为 0.8 \~ 1.0 之间，以保证 9.04 时足够模糊，0.48 时足够清晰  
  const blurAmount \= currentGSD \* 0.8;  
  // 例如：GSD=9.04 \-\> blur(7.2px) \[很糊\]; GSD=0.48 \-\> blur(0.38px) \[清晰\]

### **页面编号：4**

* **页面名称：** 无人机航拍模拟实验 (自由探究)  
* **页面类型：** Type-F (SVG动画实验页)  
* **页面主要内容与交互：**  
  1. **左侧实验面板 (SVG/Canvas交互区)：**  
     * **高度控制器：** 三个互斥按钮 \[100米\] \[200米\] \[300米\]。

     * **GSD显示框：** 初始为空或0，实验完成后显示具体数值 (单位: 厘米/像素)。
