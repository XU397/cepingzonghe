# 管理端需求文档：倒计时显示控制功能

> 版本：1.0.0
> 日期：2025-12-11
> 状态：待实施

---

## 一、需求背景

### 1.1 当前状态

目前交互评测系统的倒计时机制如下：
- 每个子模块（如无人机实验、薇甘菊实验等）有独立的倒计时
- 管理后台可以设置每个子模块的倒计时时长
- 学生端会显示倒计时时钟，学生可以看到剩余时间

### 1.2 问题描述

在某些线下考试场景中，需要：
1. **统一时间控制**：由监考人员统一控制考试时间，而非系统自动控制
2. **减少学生焦虑**：部分学生看到倒计时会产生焦虑，影响发挥
3. **灵活调整**：根据实际情况灵活延长或缩短考试时间

### 1.3 解决方案

新增"隐藏倒计时显示"功能：
- 管理后台新增开关，控制是否向学生显示倒计时
- 开关开启时，学生端不显示倒计时时钟
- 倒计时逻辑本身**不变**，只是UI不显示
- 通过线下监考人员人工控制考试时间

## 二、功能需求

### 2.1 新增配置项

在 **Flow（评测流程）配置** 或 **批次配置** 中新增以下字段：

| 字段名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `hideTimerDisplay` | Boolean | `false` | 是否隐藏学生端倒计时显示 |

### 2.2 UI设计建议

#### 位置建议
在"评测流程配置"或"批次配置"页面中，新增一个配置区域：

```
┌─────────────────────────────────────────────────────────────┐
│ 倒计时显示设置                                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ☐ 隐藏学生端倒计时                                         │
│                                                             │
│  说明：勾选后，学生端将不显示倒计时时钟，但系统倒计时        │
│  逻辑仍然生效。适用于需要线下人工控制考试时间的场景。        │
│                                                             │
│  ⚠️ 注意：启用此选项时，建议将各子模块的倒计时时间设置       │
│     为较大值（如 9999 秒），避免系统自动超时中断考试。       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 交互说明

1. **默认状态**：未勾选（显示倒计时）
2. **勾选时**：
   - 显示提示信息："启用后学生端将不显示倒计时，请通过线下方式控制考试时间"
   - 可选：弹出确认对话框
3. **保存时**：将配置保存至后端

### 2.3 配置层级

建议支持以下配置层级（优先级从高到低）：

| 层级 | 说明 | 示例 |
|------|------|------|
| Flow级别 | 针对特定评测流程 | `/flow/assessment-001` 的配置 |
| 批次级别 | 针对特定考试批次 | `batchCode=250619` 的配置 |
| 全局默认 | 系统默认值 | `false`（显示倒计时） |

**配置查找逻辑**：
```
1. 检查当前Flow是否有 hideTimerDisplay 配置
2. 如果没有，检查当前批次是否有配置
3. 如果都没有，使用默认值 false
```

## 三、接口对接

### 3.1 配置保存接口

与后端同事协商接口格式。建议方案：

**请求示例**：
```json
// POST /admin/flow/config 或 /admin/batch/config
{
  "flowId": "assessment-001",      // 或 batchCode
  "displayOptions": {
    "hideTimerDisplay": true
  }
}
```

### 3.2 配置读取接口

从后端获取当前配置状态，用于回显。

## 四、测试要点

### 4.1 功能测试

| 测试项 | 预期结果 |
|--------|----------|
| 勾选"隐藏倒计时"并保存 | 配置成功保存，再次进入页面时勾选状态保持 |
| 取消勾选并保存 | 配置更新为显示倒计时 |
| 学生端验证 | 勾选后，学生端不显示倒计时时钟 |

### 4.2 边界测试

| 测试项 | 预期结果 |
|--------|----------|
| 未配置时 | 默认显示倒计时 |
| 配置冲突时 | 按优先级规则处理（Flow > 批次 > 默认） |
